​ : 定义模型和数据库中的表之间的关联关系

urls.py 修改路由

V : 视图层(views.py 表的插入、删除、更新、查询)

​ : 定义业务逻辑

一、插入（create）

实现插入数据

def add_stu(request):

​ 第一种方式：

​ Student.objects.create(s_name=’小花’)

​ 第二种方式：

​ stu = Student()

​ stu.s_name = ‘小花’

​ stu.save()

​ return HttpResponse(‘创建学生成功’)

二、删除

实现删除数据

a.获取删除的对象，filter(条件)

b.实现删除方法，delete()

def del_stu(request):

​ Student.objects.filter(id=3).delete()

​ return HttpResponse(‘删除学生成功’)

三、更新

实现更新

def up_stu(request):

​ a.获取更新数据，filoter(条件)

​ b.实现更新方法，update（s_name = ‘妲己’）

​ 第一种方法：

​ Student.objects.filter(id=2).update(s_name=’妲己’)

​ 第二种方法：

​ stu = Student.objects.filter(id=2).first()

​ stu.s_name = ‘妲己2’

​ stu.save()

​ return HttpResponse(‘更新学生成功’)

四、查询

查询学生信息

def sel_stu(request):

1、查询所有学生信息

​ stus = Student.objects.all()

​ for stu in stus:

​ print(stu.s_name)

​ ——————————————————

​ 查询id=2的学生信息，filter()

​ stu = Student.objects.filter(id=2)first()

​ print(stu)

​ get() 取唯一的对象

​ get(条件) 条件必须成立，否则会报错

​ stu = Student.objects.get(id=2) 建议少用，一般使用filter（条件）

​ print(stu)

2、过滤出不满足条件的信息

​ stus = Student.objects.filter(s_gender=1) 所有的男生

​ stus = Student.objects.exclude(s_gender=0) 不包含女生

​ print（stus）

3、排序order_by

​ stus = Student.objects.order_by(‘id’) 升序

​ stus = Student.objects.order_by(‘-id’) 降序

​ print(stus)

4、取出对象中的某个字段

​ stus = Student.objects.all().values() 全部的字段

​ stus = Student.objects.all().values(‘s_name’,’s_age’) 只取名字和年龄字段

​ print(stus)

​ return HttpResponse(‘查询学生信息成功’)
